Introduction

  • This Slidify presentation is part of the Data Science Specialisation Capstone project on Coursera
  • The aim of this project is to develop a web application that predicts the next word in a phrase
  • This presentation aims to explain the use of the web application as well as the mechanism of prediction

The web application

  • The assingment can be found at (https://www.coursera.org/learn/data-science-project)
  • The goal is to use Natural Language Processing (NLP) to predict the next word of a phrase
  • The web application can be found at (https://littlemeadow1994.shinyapps.io/Prediction_app/)
  • The input of the web application is a word or a phrase (sequence of multiple words)
  • The ouput is the word that is predicted to be next in the phrase

How to use the web application

A caption

  1. The user enters a word or a phrase (sequence of words) at arrow 1.
  2. The prediction model will predict the next word in the phrase and displays it at arrow 2.
  3. At arrow 3 the input of the user is visualised.
  4. At arrow 4 the method of prediction is visualised.

Mechanisms of prediction

First, the input of the user is compared with three datasets:

  1. The quadgram.RData set containing four word phrases and their frequency of occurence.

  2. The trigram.RData set containing three word phrases and their frequency of occurence

  3. The bigram.RData set containing two word phrases and their frequency of occurence

Continuining , the prediction tries to find the input in one of these datasets.

Finally, the word with the highest frequency is returned.
